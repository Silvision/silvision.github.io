Eternal Return Stat Tracker


This is an Android application that tracks stats from the MOBA game, Eternal Return: Black Survival, developed and published by Nimble Neuron.
It retrieves the stats through the game's REST API at Eternal Return's Developer Portal.
Developed with Java in Android Studio, it uses Retrofit 2 as a REST client and OkHttp as the HTTP client.

Developed by: Richard Nguyen, Tim Tran, Colton Robb
