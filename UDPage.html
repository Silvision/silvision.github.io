Ultimutt Defenders


This is a Dog-themed tower defense game developed for PC and Android. Developed with C# in Unity3D, it uses Back4Apps, a backend-as-a-service platform powered by Parse Open Source, to store user data.

The game consists of 5 stages that the user can beat. More dogs (turrets) can be obtained through spending Kibble, an in-game currency, in order to beat harder stages. Along with Kibble, there are Chew Toys that can obtained to be used to upgrade the dogs.

Developed by: Richard Nguyen, Tim Tran, Georve Vo, Dat Nguyen
